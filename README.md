# MediTurno

Sistema Web de Gestión de Turnos Médicos desarrollado para la materia **Práctica Profesionalizante II**.

## Descripción

MediTurno es un proyecto académico desarrollado para la materia **Práctica Profesionalizante II**. Consiste en un sistema web para la gestión de turnos médicos, donde los pacientes pueden solicitar y consultar turnos, mientras que los administradores y médicos disponen de paneles para gestionar la información del sistema.
